# Mini RAG Frontend

React frontend for the Mini RAG application.

## Coming in Phase 7

The frontend will be built with:
- React + TypeScript
- Vite for fast builds
- Tailwind CSS for styling

## Features (Planned)
- Text input area for document ingestion
- Query box for asking questions
- Answer panel with inline citations
- Source snippets display
- Request metrics (latency, tokens, cost estimates)
